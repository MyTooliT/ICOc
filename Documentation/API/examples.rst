.. _Examples:

********
Examples
********

For code examples, please check out the `examples directory <https://github.com/MyTooliT/ICOc/tree/main/mytoolit/examples>`_:

- `Read STH Name <https://github.com/MyTooliT/ICOc/tree/main/mytoolit/examples/sth_name.py>`_: Read the name of the „first“ available STH (device id `0`).
- `Read Data Points <https://github.com/MyTooliT/ICOc/tree/main/mytoolit/examples/read_data.py>`_: Read five acceleration messages (5 · 3 = 15 values) and print their string representation (value, timestamp and message counter)
- `Store Data as HDF5 file <https://github.com/MyTooliT/ICOc/tree/main/mytoolit/examples/store_data.py>`_: Read five seconds of acceleration data and store it as HDF5 file
